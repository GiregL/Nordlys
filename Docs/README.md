# Nordlys

Nordlys is a personal assistant with multiple modules.
Its main goal is to provide myself many tools to organize and build up my own projects.

## Features

- Task board
  - Categories of tasks
  - Task status (Todo, Current, Done)
  - Priority order
- Calendar
  - Categories of events
- Money converter
- User accounts

## Architecture

- Backend API with C# and ASP.NET Core
- Frontend with React
